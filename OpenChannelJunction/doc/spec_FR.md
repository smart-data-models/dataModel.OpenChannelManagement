Description globale : **Cette entité contient une description harmonisée d'une jonction générique réalisée pour le domaine de la gestion des systèmes d'eau brute (canaux ouverts). Une jonction définit un endroit où les caractéristiques du canal changent, où deux canaux ou plus se rejoignent ou se séparent, où des quantités d'eau sont prélevées ou insérées dans le système, etc.**.    
version : 0.0.1
