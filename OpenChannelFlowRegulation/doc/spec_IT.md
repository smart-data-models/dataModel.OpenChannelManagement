Descrizione globale: **Questa entità contiene una descrizione armonizzata di una simulazione generica di una serie di strutture di regolazione indipendenti per stabilire specifiche condizioni di flusso nel sistema di trasporto. È fatta per il dominio di gestione del sistema delle acque grezze (canali aperti).  
versione: 0.0.1
