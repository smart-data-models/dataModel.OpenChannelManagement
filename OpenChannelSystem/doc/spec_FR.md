Description globale : **Cette entité contient une description harmonisée d'un système générique réalisé pour le domaine de la gestion des systèmes d'eau brute (canaux ouverts). Cette entité représente soit un système composé de différents éléments (par exemple, des canaux, des jonctions, des sections transversales, etc.), soit un simple élément (par exemple, un SluiceGate).    
version : 0.0.1
